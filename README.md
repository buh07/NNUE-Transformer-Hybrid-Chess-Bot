# NNUE Transformer Hybrid Chess Bot

A hybrid chess engine combining NNUE (Efficiently Updatable Neural Network) and Transformer architectures for enhanced chess position evaluation and move prediction.

## Features

- Hybrid architecture combining NNUE efficiency with Transformer expressiveness
- Chess position evaluation and analysis
- Python implementation using the `chess` library

## Setup

1. Create a virtual environment:
```bash
python3 -m venv chess_env
source chess_env/bin/activate  # On Windows: chess_env\Scripts\activate
```

2. Install dependencies:
```bash
pip install chess
```

## Project Structure

- `ChessGame.py` - Main chess game implementation
- `hybrid_nnue_transformer_implementation.md` - Implementation documentation
- `chess_env/` - Python virtual environment (not tracked in git)

## Usage

```bash
python ChessGame.py
```
